# bank_system
## 💻 MINI BANKING SYSTEM (Python)
This project is a simple banking system developed in Python that simulates basic operations such as account creation, authentication, and transaction management.
--------------------------------------------------------------------------------------------------------------------------------------------------
# 🚀 Features
- Create multiple bank accounts
- Secure login system with a maximum of 3 attempts
- Check account balance
- Deposit and withdraw money
- Data persistence using CSV files
- Efficient account management using Pandas

# 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- Classes and encapsulation
- Data handling with Pandas DataFrames
- File handling (CSV)
- Loops and conditional logic

# 📂 How It Works
1. User creates one or more accounts
2. Account data is stored in a CSV file
3. User logs in using name, bank, and PIN
4. After authentication, a menu allows:
- Viewing balance
- Depositing money
- Withdrawing money
5. Changes are saved back to the CSV file

# ⚠️ Notes
- PIN is required for secure operations
- Account access is blocked after 3 failed login attempts
- Data is stored locally in accounts.csv
